# Tools

This directory contains scripts and tools for maintaining and working with the prompts in this repository.

## Subdirectories

- **[prompt_tools](prompt_tools/README.md)**: A collection of utilities for refining, cleaning, or reorganizing prompts.
- **[scripts](scripts/README.md)**: Utility scripts for repository maintenance, validation, and workflow execution.

## Getting Started

To explore the available scripts, visit the `scripts/` directory. For advanced prompt engineering tools, check `prompt_tools/`.
