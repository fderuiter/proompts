# Architecture Review Prompts

This folder contains prompts for analyzing and reviewing a codebase's architecture, dependencies, and overall quality, as well as guides for specific architectural patterns.

## Prompts

- `01_dry_codebase_analysis.prompt.yaml`: Analyzes code for Don't Repeat Yourself (DRY) violations.
- `02_solid_codebase_analysis.prompt.yaml`: Analyzes code for SOLID principle adherence.
- `03_maintainability_codebase_analysis.prompt.yaml`: Analyzes code for maintainability and technical debt.
- `04_hexagonal_architecture_principles.prompt.yaml`: Explains the core philosophy, skeleton, and benefits of Hexagonal Architecture.
- `05_hexagonal_architecture_implementation.prompt.yaml`: Guide for implementing Hexagonal Architecture, focusing on data flow and dependency inversion.
- `06_hexagonal_architecture_review.prompt.yaml`: Analyze code for adherence to Hexagonal Architecture principles, identifying layer violations.
