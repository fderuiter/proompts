# Architecture Review Prompts

This folder contains prompts for analyzing and reviewing a codebase's architecture, dependencies, and overall quality.

## Prompts

- `01_dry_codebase_analysis.prompt.yaml`: Analyzes code for Don't Repeat Yourself (DRY) violations.
- `02_solid_codebase_analysis.prompt.yaml`: Analyzes code for SOLID principle adherence.
- `03_maintainability_codebase_analysis.prompt.yaml`: Analyzes code for maintainability and technical debt.
