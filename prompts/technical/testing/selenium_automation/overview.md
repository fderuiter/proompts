# Selenium Automation Overview

## Prompts
- **[Framework Implementation: Data-Driven Testing](data_driven_selenium.prompt.yaml)**: Utilize data providers to execute the same test logic with multiple sets of data.
- **[Advanced Design Patterns: Fluent Interface](fluent_interface_selenium.prompt.yaml)**: Extend Page Objects with method chaining to create a more readable test API.
- **[Framework Best Practices: Locator Strategy](locator_optimization.prompt.yaml)**: Transition from brittle XPaths to robust locators like ID or CSS selectors.
- **[Project Configuration: Maven Setup](maven_selenium_setup.prompt.yaml)**: Set up a Maven project structure and pom.xml file with necessary Selenium client libraries.
- **[Execution Optimization: Parallel Testing](parallel_execution.prompt.yaml)**: Configure the automation suite to execute multiple tests simultaneously.
- **[Architecture Design: Page Object Model](pom_implementation.prompt.yaml)**: Implement the Page Object Model pattern to separate UI locators from test logic.
- **[Selenium Migration: Script Conversion](script_conversion.prompt.yaml)**: Translate recorded browser actions (Selenese) from Selenium IDE into structured Java test scripts.
- **[Cross-Browser Infrastructure: Selenium Grid](selenium_grid_setup.prompt.yaml)**: Configure a Selenium Grid Hub and Nodes to distribute test execution.
- **[Test Environment: Python & Selenium Base](selenium_python_setup.prompt.yaml)**: Install the Selenium Base framework and environment using the Python package manager.
- **[Reporting and Maintenance: Custom Reports](selenium_reporting.prompt.yaml)**: Integrate reporting libraries and screenshot utilities to capture execution evidence.
- **[Synchronization Strategy: Explicit Waits](selenium_waits.prompt.yaml)**: Replace brittle Thread.sleep() calls with dynamic Explicit or Fluent waits.
- **[Security Testing: OWASP ZAP Integration](selenium_zap_integration.prompt.yaml)**: Configure Selenium to route traffic through the OWASP ZAP proxy for security scanning.
- **[Driver Configuration: WebDriver Initialization](webdriver_initialization.prompt.yaml)**: Initialize and configure WebDriver instances with specific browser options.
