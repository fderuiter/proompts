# Python Overview

## Prompts
- **[Advanced Python Testing](advanced_python_testing.prompt.yaml)**: A comprehensive guide to Python testing, covering Pytest fixtures, Property-Based Testing (Hypothesis), and Mutation Testing.
- **[Principal Python Developer](principal_python_developer.prompt.yaml)**: A Principal Engineer's guide to Python development, focusing on architecture, decoupling, robustness, concurrency, and observability.
- **[Python Concurrency Mastery](python_concurrency_mastery.prompt.yaml)**: A Principal-level guide to mastering Python concurrency, focusing on AsyncIO, Structured Concurrency, and Multiprocessing safety.
- **[Python Hexagonal Architecture](python_hexagonal_architecture.prompt.yaml)**: A Principal-level guide to decoupling Python systems using Hexagonal Architecture, Protocols, and Dependency Injection.
- **[Python Performance Optimization](python_performance_optimization.prompt.yaml)**: A Senior-level guide to optimizing Python code, focusing on profiling, memory management, and GIL workarounds.
- **[Senior Python Developer](senior_python_developer.prompt.yaml)**: A Senior Developer's guide to Python execution, focusing on idiomatic code, maintainability, and code stewardship.
