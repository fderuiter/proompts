- **[Test Architect (Automated Testing)](add_tests.prompt.yaml)**: Generates comprehensive unit and integration tests for provided code, focusing on edge cases, reliability, and clean code practices.
- **[Architecture Flow & Diagram Architect](architecture_flow.prompt.yaml)**: A Principal System Architect's guide to tracing request lifecycles, identifying bottlenecks, and generating Mermaid diagrams.
- **[Bug Finder & Fixer (OpenAI Codex)](bug_fix.prompt.yaml)**: Reproduce and resolve a bug within the specified package or module.
- **[Continuous Integration & Delivery (DevOps Architect)](ci_cd.prompt.yaml)**: Design and implement secure, efficient, and scalable CI/CD pipelines for AI-integrated applications.
- **[Code Review Assistant (Aegis Security)](code_review.prompt.yaml)**: Conduct a comprehensive security-focused code review, identifying vulnerabilities, logic flaws, and style issues with a structured report.
- **[Codebase Testing Plan](codebase_testing_plan.prompt.yaml)**: As a Distinguished Quality Engineer, generate a comprehensive testing strategy and implementation roadmap for an existing codebase. This includes risk analysis, tooling selection, and a phased rollout plan aligned with modern CI/CD practices.
- **[GitHub Custom Agent Creator](create_github_agent.prompt.yaml)**: Expertly craft configuration files for GitHub Custom Agents with strict YAML frontmatter and structured Markdown instructions.
- **[DevEx Documentation Architect](docs_and_onboarding.prompt.yaml)**: A Senior Developer Experience Engineer's guide to creating world-class documentation, onboarding paths, and architectural records.
- **[Principal Architect Task Execution](principal_architect.prompt.yaml)**: A Principal Architect persona for executing tasks from TODO.md with strict adherence to SOLID, DRY, YAGNI, and KISS principles.
- **[Project Init & Skeleton (Construct Architect)](project_init.prompt.yaml)**: A Principal Cloud-Native Architect's blueprint for initializing secure, scalable, and 12-Factor compliant project skeletons.
- **[Retrieval-Augmented Answer Composer](rag_composer.prompt.yaml)**: Provide concise answers using only supplied knowledge-base files.
- **[Refactoring Architect](refactoring_suggestions.prompt.yaml)**: A Principal Software Architect's guide to surgical refactoring, focusing on decoupling, testability, and debt remediation.
- **[Security Vulnerability Hunt (Aegis)](security_vulnerability.prompt.yaml)**: Locate and fix memory-safety vulnerabilities in C/C++ code with a structured analysis.
- **[Tooling & Quality Gates (DevEx Architect)](tooling_and_quality.prompt.yaml)**: A Distinguished Developer Experience Engineer's guide to enforcing code quality, strict typing, and "fail-fast" CI/CD pipelines.
- **[UI Tweak & Verification (Aegis Security)](ui_fix.prompt.yaml)**: Resolve a minor UI regression and confirm the fix with build or test steps, ensuring accessibility and security.
