# Software Engineering Prompts

This directory contains prompts related to software engineering, organized into two layers:

- **Lifecycle (`lifecycle/`)**: Strategic prompts for managing the entire software development lifecycle, from product conception to epic definition and review.
- **Tasks (`tasks/`)**: Tactical prompts for specific engineering tasks like project initialization, testing, CI/CD, and refactoring.

## Structure

### Lifecycle
Focuses on the *process* and *strategy*.
- Product Briefs
- Technical Implementation Plans
- Coding Session Guidelines
- Project Reviews

### Tasks
Focuses on the *execution* and *tactics*.
- Scaffolding & Initialization
- Testing Plans
- CI/CD Setup
- Code Reviews
- Refactoring
