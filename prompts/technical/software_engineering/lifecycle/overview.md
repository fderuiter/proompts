# Lifecycle Overview

## Categories
- [Agentic Coding Workflow/](agentic_coding_workflow/overview.md)

## Prompts
- **[Coding Session Guidelines](coding_session_guidelines.prompt.yaml)**: Provide step-by-step guidance for running productive coding sessions.
- **[E2E Test Discovery Lifecycle Template](e2e_test_discovery.prompt.yaml)**: Provide a system prompt template that guides an LLM to analyse a codebase and generate a comprehensive end-to-end test plan.
- **[Folder and Module Organization](folder_module_organization.prompt.yaml)**: Provide a detailed plan for restructuring a Python codebase into clearer, feature-based modules.
- **[Project Memory Notes](project_memory.prompt.yaml)**: Maintain a running log of architectural decisions and contextual notes for the project.
- **[Project Review Checklist](project_review.prompt.yaml)**: Verify completion of a coding project before finalizing.
- **[Reflexion Agent Bug Patch](reflexion_agent_bug_patch.prompt.yaml)**: Locate and fix a bug using a structured reflexion workflow.
- **[RequirementsBot Prompt](requirements_prompt.prompt.yaml)**: Guide an AI assistant to inspect a repository and generate a complete `REQUIREMENTS.md` file.
- **[Technical Implementation Plan](technical_implementation_plan.prompt.yaml)**: Detail the architecture, dependencies, and steps required to implement a project.
- **[To-Do List Template](todo_generation.prompt.yaml)**: Track pending and completed development tasks.
