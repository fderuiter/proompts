# Prompt Engineering Overview

## Prompts
- **[Adversarial Prompt Robustness Tester](adversarial_prompt_robustness_tester.prompt.yaml)**: Acts as a Principal AI Red Teamer to systematically stress-test draft prompts against adversarial injections, jailbreaks, and logical fallacies, providing architectural recommendations for hardening.
- **[Vector Prompt Editor-in-Chief](vector_prompt_editor_in_chief.prompt.yaml)**: Reviews draft prompts, eliminates generic phrasing, and elevates them with specific personas, precise constraints, and contextual framing.
