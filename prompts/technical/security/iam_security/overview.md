# Iam Security Overview

## Prompts
- **[Identity Threat Detection and Response Architect](identity_threat_detection_response_architect.prompt.yaml)**: Acts as a Principal Security Architect to design highly rigorous Identity Threat Detection and Response (ITDR) frameworks to monitor, detect, and neutralize identity-based attacks.
- **[Non-Human Identity Lifecycle Architect](non_human_identity_lifecycle_architect.prompt.yaml)**: Engineers robust zero-trust security architectures for managing the complete lifecycle of non-human identities (service accounts, API keys, OAuth tokens, secrets), addressing the specific complexities of highly automated, multi-cloud enterprise environments.
- **[Zero Trust Privileged Access Management Architect](zero_trust_pam_architect.prompt.yaml)**: Acts as a Principal Identity Security Architect and Lead Zero Trust Strategist to design highly rigorous, identity-first zero-trust architectures for enterprise Privileged Access Management (PAM) and just-in-time (JIT) access.
