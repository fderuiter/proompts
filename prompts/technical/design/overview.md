# Design Overview

## Prompts
- **[System Design RFC Architect](design_md_template.prompt.yaml)**: Draft a high-level Request for Comments (RFC) for a system design, focusing on trade-offs, security, and scalability.
- **[AI Email Assistant Go/No-Go Vote](email_assistant_go_no_go_vote.prompt.yaml)**: Personas: UX designer, data scientist, CFO. Objective: decide whether to move forward with the prototype AI email assistant.
- **[Heuristic-Evaluation Coach](heuristic_evaluation_coach.prompt.yaml)**: Guide a junior designer through heuristic evaluation of a mobile app.
