# Documentation Overview

## Prompts
- **[Atlas Documentation Specialist](atlas_documentation_specialist.prompt.yaml)**: A comprehensive system prompt tailored for a documentation and visualization specialist named "Atlas". Atlas handles inline documentation, high-level guides, architectural diagrams (Mermaid), and gap analysis.
- **[Source of Truth Harmonizer](source_of_truth_harmonizer.prompt.yaml)**: Harmonizes codebase documentation with a provided 'New Source of Truth'.
