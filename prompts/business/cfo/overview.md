# Cfo Overview

## Categories
- [**Cfo Workflow**](cfo_workflow/overview.md)

## Prompts
- **[Board Deck Narrative Generation](board_deck_narrative.prompt.yaml)**: Draft the 'CFO Commentary' slide for a Board meeting.
- **[Earnings Call Script Prep](earnings_call_script_prep.prompt.yaml)**: Prepare for tough analyst questions on an earnings call.
- **[Investor FAQ Generation](investor_faq_generation.prompt.yaml)**: Generate an FAQ for a bearish investor based on press release and 10-K.
- **[Liquidity Stress Test](liquidity_stress_test.prompt.yaml)**: Run a stress test on cash flow forecast assuming a drop in collections.
- **[M&A Target Evaluation](ma_target_evaluation.prompt.yaml)**: Evaluate a potential acquisition target based on financial statements.
- **[Net Present Value Socratic Tutor](npv_tutor.prompt.yaml)**: Guide the learner to derive and apply the NPV formula through short Socratic questioning.
- **[Regulatory Compliance Summary](regulatory_compliance_summary.prompt.yaml)**: Summarize key financial disclosure changes for a specific regulation.
- **[Scenario Modeling & Sensitivity Analysis](scenario_modeling_sensitivity.prompt.yaml)**: Create three financial scenarios (Conservative, Base, and Aggressive) based on historical data and market trends.
- **[Budget Variance Analysis](variance_analysis.prompt.yaml)**: Identify top variances in a budget-to-actuals report and draft explanations for the Board.
