# Device Specifics Overview

## Prompts
- **[Automated Image Assessment System 510(k)](automated_image_assessment_system_510_k.prompt.yaml)**: Draft a detailed device description for an automated image assessment system for microbial colonies.
- **[Carrier Screening System 510(k)](carrier_screening_system_510_k.prompt.yaml)**: Compile technical information for an autosomal recessive carrier screening gene mutation detection system.
- **[Clinical Chemistry Device Classification](clinical_chemistry_device_classification.prompt.yaml)**: Identify classification and regulatory requirements (general/special controls) for a clinical chemistry device.
- **[Design Verification for BCR-ABL Tests](design_verification_for_bcr_abl_tests.prompt.yaml)**: Outline design verification and validation requirements for a BCR-ABL quantitation test.
- **[iCGM Clinical Testing Strategy](icgm_clinical_testing_strategy.prompt.yaml)**: Draft a clinical study plan to demonstrate accuracy for an iCGM system.
- **[NGS Tumor Profiling Documentation](ngs_tumor_profiling_documentation.prompt.yaml)**: Develop documentation supporting the clinical significance of mutations in an NGS-based tumor profiling panel.
- **[Special Controls Labeling Compliance](special_controls_labeling_compliance.prompt.yaml)**: Generate mandatory labeling content, including warnings and limitations, for HCV antibody tests.
- **[Zika Virus Reagent Study Design](zika_virus_reagent_study_design.prompt.yaml)**: Draft a protocol for analytical performance studies to validate Zika virus serological reagents.
