# Adherence Overview

## Prompts
- **[ALCOA-C Data Integrity Checklist](alcoa_c_data_integrity_checklist.prompt.yaml)**: Ensure data integrity following ALCOA-C principles.
- **[DHT Integration Regulatory Checklist](dht_integration_checklist.prompt.yaml)**: Review FDA guidance for digital health technology (DHT) integration and validation.
- **[ICH E9(R1) Estimand Builder](estimand_framework_builder.prompt.yaml)**: Construct a primary estimand description following the ICH E9 (R1) framework.
- **[Human Factors/Usability Summary](human_factors_usability_summary.prompt.yaml)**: Summarize usability testing results to demonstrate minimized use-related risks.
- **[Imaging Endpoint Process Standards Checklist](imaging_endpoint_process_standards.prompt.yaml)**: Review FDA guidance on imaging endpoints and create process checklists.
- **[Informed Consent Exception (Emergency)](informed_consent_exception_emergency.prompt.yaml)**: Draft IRB documentation for an exception from informed consent in emergency research.
- **[Intended Use and Indications for Use Alignment](intended_use_and_indications_for_use_alignment.prompt.yaml)**: Review 510(k) drafts to ensure 'Intended Use' and 'Indications for Use' are verbatim and consistent.
- **[Multiple Endpoints Regulatory Strategy](multiple_endpoints_guidance_review.prompt.yaml)**: Review and summarize FDA guidance on multiple endpoints and multiplicity strategies.
- **[Off-Label Information Dissemination](off_label_information_dissemination.prompt.yaml)**: Prepare a mandatory disclosure statement for disseminating peer-reviewed articles on unapproved uses.
- **[RWE Regulatory Framework Summary](rwe_framework_summary.prompt.yaml)**: Review the FDA Real-World Evidence (RWE) Framework and fitness-for-use criteria.
- **[Shelf-life Study Rationale](shelf_life_study_rationale.prompt.yaml)**: Draft a rationale for correlating accelerated aging data with real-time requirements.
