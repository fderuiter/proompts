# Music Therapy Workflow Overview

## Prompts
- **[Compassionate Analyst](01_compassionate_analyst.prompt.yaml)**: Deconstructs user venting into actionable psychological data.
- **[ISO Strategist](02_iso_strategist.prompt.yaml)**: Plans the therapeutic journey using the ISO Principle.
- **[Sonic Architect](03_sonic_architect.prompt.yaml)**: Translates the emotional arc into concrete music theory and production choices.
- **[Lyricist](04_lyricist.prompt.yaml)**: Writes the actual lyrics using the themes and structure.
