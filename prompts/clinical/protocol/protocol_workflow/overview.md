# Protocol Workflow Overview

## Prompts
- **[Clinical-Trial Protocol Creator](01_clinical_trial_protocol_creator.prompt.yaml)**: Generate a full clinical-trial protocol from a one-page summary sheet.
- **[Ultimate SOP Architect](02_ultimate_sop_architect.prompt.yaml)**: Create a clear, regulation-compliant standard operating procedure.
- **[Protocol Reviewer and Gap-Analysis Coach](03_protocol_reviewer_gap_analysis_coach.prompt.yaml)**: Evaluate a clinical-trial protocol for patient experience, site feasibility, and regulatory completeness.
- **[Protocol Section Refinement](04_protocol_section_refinement.prompt.yaml)**: Improve the eligibility criteria section of an IVD performance trial protocol.
