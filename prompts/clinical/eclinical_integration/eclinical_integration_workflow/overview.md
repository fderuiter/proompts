# Eclinical Integration Workflow Overview

## Prompts
- **[Architect the Integration Blueprint](01_architect_integration_blueprint.prompt.yaml)**: Provide a structured plan for integrating site EHR systems with the sponsor's EDC and CTMS.
- **[Data Mapping and Transformation Playbook](02_data_mapping_transformation_playbook.prompt.yaml)**: Provide a repeatable workflow for mapping JSON FHIR bundles to SDTM-compliant tables.
- **[Regulatory and Validation Checklist](03_regulatory_validation_checklist.prompt.yaml)**: Compile a compliance checklist for digital trial data integrations.
