# High-Dimensional Analysis

This directory contains advanced prompt templates for high-dimensional statistical modeling, focusing on scenarios where the number of predictors ($p$) exceeds the sample size ($n$).

## Topics Covered
*   Penalized regression methodologies (Lasso, Ridge, Elastic Net, SCAD, MCP).
*   Variable selection consistency and oracle properties.
*   Optimization algorithms for sparse models (Coordinate Descent, ADMM, LARS).
*   High-dimensional inference and post-selection inference.

These templates are designed for Principal Statisticians and Lead Quantitative Methodologists, strictly enforcing rigorous mathematical formulations and algorithmic design using LaTeX.
