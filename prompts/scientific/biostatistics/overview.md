- **[Generate & QC Submission-Ready TLFs](submission_ready_tlfs.prompt.yaml)**: Produce validated tables, listings, and figures (TLFs) ready for regulatory submission.
- **[Time-to-Event Analysis Coach](time_to_event_analysis_coach.prompt.yaml)**: Guide a junior analyst through performing a time-to-event analysis.
- **[Universal Template-Table Prompt](universal_template_table_prompt.prompt.yaml)**: Create a formatted safety table from an ADaM ADAE dataset using either R or SAS.
