# Time Series Overview

## Prompts
- **[markov_switching_var_architect](markov_switching_var_architect.prompt.yaml)**: Formulates rigorous Markov-Switching Vector Autoregression (MS-VAR) models for analyzing non-linear macroeconomic dynamics and regime changes.
- **[mixed_frequency_dynamic_factor_nowcasting_architect](mixed_frequency_dynamic_factor_nowcasting_architect.prompt.yaml)**: Formulates rigorous Mixed-Frequency Dynamic Factor Models (MF-DFM) for high-frequency macroeconomic nowcasting to handle ragged-edge data and state-space estimation.
- **[structural_vector_autoregression_architect](structural_vector_autoregression_architect.prompt.yaml)**: Formulates rigorous Structural Vector Autoregression (SVAR) models for macroeconomic shock identification, providing identification schemes, impulse response functions (IRFs), and variance decompositions.
