- **[transfinite_induction_well_ordering_architect](transfinite_induction_well_ordering_architect.prompt.yaml)**: Acts as a Principal Set Theorist to rigorously formulate multi-step proofs using transfinite induction and well-ordering principles over arbitrary ordinal and cardinal structures.
